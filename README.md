# 本项目初衷

没有发现有哪个网站可以找出：例如：X 级车中百米散布最小的车，因此有了这个项目

# 步骤1
注册成为 WG 的开发者

# 步骤2
- 在 WG 的开发者平台创建应用（替换本项目中的 application_id）
- 在 WG 的开发平台配置你的程序出口 IP 地址白名单

# 步骤3 
执行 Main 函数，复制控制台打印的全量坦克数据
在企业微信中创建智能表格（配置好表头的对应关系）
